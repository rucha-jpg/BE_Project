# Problem Statement

Public facilities lack an integrated system that provides real-time information on restroom stall occupancy and cleanliness. Existing solutions typically monitor either occupancy or air quality independently, resulting in inaccurate usage detection and inefficient maintenance practices.

This limitation leads to improper resource utilization, delayed cleaning schedules, and reduced user satisfaction. Therefore, there is a need for a unified system that combines IoT and machine learning to enable accurate occupancy detection, real-time air quality monitoring, and predictive facility management.
